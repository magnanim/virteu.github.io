REQUESTS FROM DATA SUBJECTS

As a data subject, you can request one of the following actions, that will be evaluated based on its impact on the research study:

  • Get the tweets produced by you that we have stored in our database.
  • Request the removal of (some of) the tweets produced by you that we have stored in our database.
  • Opt out from this research, that is, disallow us from storing tweets produced by you in our database from now on, in the context of this research project.

If you want to do so, you have to:

  1. Follow the Twitter account @eu_virt using the Twitter user whose data has been collected (this is needed to make sure that the request comes from you).
  2. If you want to receive a copy of all the tweets produced by you that we have stored in our database: send a private message to the @eu_virt account writing "Get"
  3. If you want to request the removal of the tweets produced by you that we have stored in our database: send a private message to the @eu_virt account writing "Remove"
  4. If you want us not to store tweets produced by you from now on, but you allow us to keep the tweets we have collected so far, send a private message to the @eu_virt account writing "Opt out"

You will then be notified about the receipt and processing status of the request, and can at that point unfollow the Twitter account.
